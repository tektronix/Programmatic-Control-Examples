# Using Bipolar Output Capabilities

Generates a linear voltage sweep while taking measurements.

The default configuration performs a voltage sweep on module 2, channel 2, starting at 0.5 V and stopping at 5 V, with increments of 0.5 V and a current limit set to 1 A. Could outputs sweep setting and measurement at every sweep point.

## Required Modules
1 x MPSU50-2ST

## Available Languages
* TSP
* Python

## Instructions
1. Adjust settings as needed and run code.
