# Demonstrating Overcurrent Protection (OCP)

Configures PSU to intentionally trigger and demonstrate OCP.

Settings assume a 50 ohm load gets shorted during operation, triggering OCP to shut the power supply off. 

## Required Modules
1 x MPSU50-2ST

## Available Languages
* TSP
* Python

## Instructions
1. Adjust settings as needed and run code.
