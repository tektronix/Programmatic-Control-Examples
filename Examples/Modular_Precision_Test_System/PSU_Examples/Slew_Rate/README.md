# Controlling Slew Rate

Toggles outputs using different slew rates.

This code demonstrates the difference in slew rates of the unit and illustrates how the slew rate can be applied in various situations. The current output on the oscilloscope displays a comparison between a slew rate of 1 and a slew rate of 10,000, highlighting the effectiveness of the slew rate command.

## Required Modules
1 x MPSU50-2ST

## Available Languages
* TSP
* Python
