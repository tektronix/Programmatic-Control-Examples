# BJT Vce-Ic Characterization

Two channels are used to perform a voltage and current sweep for a Vce-Ic characterization of a BJT. Measurements are recorded

NOTE: There are two functions provided. “OutputCurve_VceIc_TriggerInOneSlot” handles 2 channels on one SMU and “OutputCurve_VceIc_TwoTriggers” handles using two channels on different slots.

## Required Modules
1 x MSMU60-2

## Available Languages
* TSP
* Python
